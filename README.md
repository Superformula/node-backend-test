# User's API
Provides endpoints to Create/Read/Update/Delete users.  See the detailed [instructions](problem-description.md)

## Development

### Prerequisites
- Node 10+
- NPM
- Yarn
- Git
- GitBash

### Install serverless
`npm -g install serverless`

### Start local service
`yarn start`  
Verify: `curl localhost:3000/users/1`
